# 🔐 Role-Based Login System (Angular)

A modern Angular application implementing **Role-Based Access Control (RBAC)** with authentication, route protection, and dynamic UI rendering based on user roles.

---

## 🚀 Features

* 🔑 User Authentication (Login/Logout)
* 🛡️ Role-Based Access Control (Admin/User)
* 🚧 Route Guards for protected navigation
* 📊 Dynamic Dashboard based on user role
* 🔄 Session handling (localStorage)
* 🎯 Clean and modular Angular architecture

---

## 🛠️ Tech Stack

* **Frontend:** Angular, TypeScript, HTML, CSS
* **State Management:** Services
* **Routing:** Angular Router + Guards

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/sudheee-dev/Role-based-login-angular.git
```

2. Navigate to project folder

```bash
cd Role-based-login-angular
```

3. Install dependencies

```bash
npm install
```

4. Run the application

```bash
ng serve
```

5. Open in browser

```
http://localhost:4200
```
